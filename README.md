# ZobShop

Administrator:
- Create product
- Edit product
- Delete product
- Manage users
- Add product reviews

User:
- Browse products
- Add to cart
- Manage profile
- Give review about product
- Finish order
- Comment on product reviews

Products have:
 - Name
 - Maker
 - Category
 - Quantity 
 - Price
 - Volume / Size
 - Reviews
 - Rating
 
 Category:
  - Name
  
 Review:
  - Comments
  - Rating
  - Product
  - Product rating
  - Text
  - Date
  - Author
  
  Comment:
   - Date
   - Review
   - Author
   - Text
